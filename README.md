# Marshian

### Built with React CLI, displaying data from SpaceX API

### To run the project

- \$ git clone https://github.com/habin-isa/marshian.git
- \$ cd marshian
- \$ yarn
- \$ yarn start

### Examples contained

- Styled components
- Jest testing
- Proptypes

### Tech

- React CLI
- Pretty-quick
- Styled components
- Jest
- Axios
- Babel/EsLint
